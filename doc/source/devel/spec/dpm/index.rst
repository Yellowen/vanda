Dina Package manager
====================
The most important part of Dina platform is its package manager system. Dina Package Manager of DPM for short is a powerful debian like package manager that allow user to
select and install his/her desire packages as easier as possible. The idea behind DPM was to create a package manager for Dina platform just like debian dpkg, so developers
can easily work with it and learning DPM internals takes no time.
