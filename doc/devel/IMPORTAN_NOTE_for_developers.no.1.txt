#+++  comment dar morede ezafe shodane chizi

3. comment gozari code yadetun nare .

4. lotfan az namhayi baraye variable haye mohem va function ha o class ha estefade konin ke ba mani bashan  ( khodetun manzuramo motevajeh mishin dige ;-) )

5. az estefade az function e " print " jedan khod dari konin va bezash ( baraye testo in harfa ) az function e dassert () estefade konin . in function tu module dina.core.dev tarif shode

6. vaghti commit mikonin yadetun bashe ke string i ke be onvane tozih vared mikonin bayad monasebe taghirati ke dadin bashe.

7. say konin beyn block ha va ghesmat hayi az code ke be ham marbut mishan 2 ya 3 khat fasele bashe.

8. say konin har commiti ke mikonin arzeshe commit dashte bashe yeni baad az anjame movafaghe amali commit konin .
